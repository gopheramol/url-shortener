# url-shortener
This is a simple URL shortener service written in Go.

## Stack
### Golang
### Clean code architecture
### Domain driven architecture 

## API :
### Encode
http://localhost:8080/encode

### Decode
http://localhost:8080/decode/phcjoIEV

```make run```

```make build```
